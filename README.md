is-int [![Build Status](https://travis-ci.org/RivalNick/is-int.svg?branch=master)](https://travis-ci.org/RivalNick/is-int)
============================================================

> Check if a variable is an integer.

## Install
```shell
npm install is-int
```

## Usage
```javascript
var isInt = require('is-int');

isInt(400);	// => true

isInt("hello");	// => true
```
